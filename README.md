## What is the Unit Converter?
**Unit Converter** is a simple web application that converts values between **metric** and **imperial** units for:
- Length
- Volume
- Weight

The project focuses on JavaScript logic, user input handling, and clear UI presentation.

## Features
- Convert between metric and imperial units
- Supports length, volume, and weight conversions
- Simple, user-friendly interface
